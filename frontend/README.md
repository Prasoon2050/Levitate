# Levitate - Next.js Migration

This project has been migrated from Vite to Next.js 15 (App Router).

## Getting Started

First, run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Project Structure

- `app/`: Contains the main application code (App Router).
- `components/`: Contains all React components.
  - `components/layout/`: Structural components like Navbar and Footer.
  - `components/sections/`: Landing page sections (Hero, Features, etc.).
- `public/`: Static assets.

## Scripts

- `npm run dev`: Starts the development server.
- `npm run build`: Builds the application for production.
- `npm start`: Starts the production server.
- `npm run lint`: Runs ESLint.
